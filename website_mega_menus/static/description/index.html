Website Mega Menu

Mega menu is a two-dimensional drop-down interface that is triggered by the user hovering over a link. All the options available are shown by dropdown under a specified category which contains a group of related products.

A promotional tool called Odoo Website Mega Menu allows customers to navigate easily on Website and hence increases the sale. It helps to improve user-interface by making the website more interactive and attractive.

Website Dropdown View

The Mega Menu can be placed on the website as a separate header.

The hover on feature allows the Mega Menu to appear on when the customer want to use it.

The mega menu can be created with multi-level category tree.

You can also put the Mega menu inside a single tab.

The customer can use the mega menu by clicking on the tab.

Website Fly-Out View

Another view provided with the module is the fly-out view or horizontal view.

The flyout Mega menu appears from left to right on the website page.

The fly out view can also be configured with multi-levelled category views.

The Odoo user can create the Mega menu in the Odoo.

Images and icons can also be added to the mega menu as per the need.
